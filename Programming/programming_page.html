Biomedical Analysis Software Suite (BASS)

The Biomedical Analysis Software Suite (BASS) comprises a Python Module and accompanying IPython notebooks. Each notebook is a variant of a data analytics pipeline that detects and measure events within a time series. The underlying architecture makes it easy to customize the parts of your detection pipeline as well as pick from a menu of analysis options.

Features

  1. Signal Processing: Detrending (linear subtraction), Bandpass filters, Savitzky-Golay Filter
  2. Event Detection: Peaks and boundaries (Bursts)
  3. Event Measurements: Peak amplitude, Peak-Peak Intervals, Duration, Interburst interval, Total Cycle
  4. Time, Peaks per Burst, Intraburst Frequency, Burst Area, Attack, Decay.
  5. Event Analysis: Descriptive Statistics, Histogram Entropy, Approximate Entropy, Sample entropy, Poincare Plots, Moving averages, Power Spectral Density, Frequency.

Neuron Modelling

A set of python classes based off of [BRS], [Toporikova-Butera], [Yan et al.] used to model single neurons. The parameters of each neuron are fully customizable, but also have reasonable defaults, so that the classes can be run right out of the box.
